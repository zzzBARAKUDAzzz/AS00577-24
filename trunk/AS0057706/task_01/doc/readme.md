## Цель работы
Смоделировать температуру управляемого объекта, поведение которого описывается дифференциальным уравнением, преобразованным в дискретные линейную и нелинейную модели. Написать программу на языке С++ для симуляции этих моделей.

---

## Теоретическая часть

Исходное дифференциальное уравнение изменения температуры объекта:
$$\frac{dy(\tau)}{d\tau} = \frac{u(\tau)}{C} + \frac{Y_0 - y(\tau)}{RC}$$

После математических преобразований в дискретное время были получены следующие модели:

* **Линейная модель:** 
  $$y_{\tau+1} = a y_{\tau} + b u_{\tau}$$

* **Нелинейная модель:** 
  $$y_{\tau+1} = a y_{\tau} - b y_{\tau-1}^2 + c u_{\tau} + d \sin(u_{\tau-1})$$

**Где:**
* $\tau$ — дискретные моменты времени ($1, 2, 3 \dots n$);
* $y(\tau)$ — температура объекта;
* $u(\tau)$ — входное тепловое воздействие;
* $Y_0$ — комнатная температура;
* $a, b, c, d$ — константы математической модели.
