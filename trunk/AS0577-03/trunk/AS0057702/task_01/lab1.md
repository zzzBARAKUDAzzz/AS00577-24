Тема

Моделирование управляемого объекта.
Цель работы

Разработать программу на языке C++, которая моделирует изменение температуры объекта по линейной и нелинейной дискретным моделям.
Используемые технологии

    C++
    CMake
    Git
    GitHub
    Visual Studio Code

Математическая модель

Линейная модель:

y(t + 1) = a * y(t) + b * u(t)

Нелинейная модель:

y(t + 1) = a * y(t) - b * y(t - 1)^2 + c * u(t) + d * sin(u(t - 1))
Описание программы

В программе реализованы две функции:

    linearModel() — рассчитывает температуру по линейной модели.
    nonlinearModel() — рассчитывает температуру по нелинейной модели.

Программа выполняет моделирование на 20 шагах и выводит результаты в консоль.
Сборка проекта

Запуск

Для Windows:

Вывод

В результате работы программы выводится таблица значений температуры для линейной и нелинейной моделей.
