## Описание задачи

Программа моделирует изменение температуры объекта управления на основе двух разностных уравнений — линейного и нелинейного. Объект описывается температурой y(t), на вход подаётся управляющее воздействие u(t) (подводимое тепло).

**Линейная модель:**  
y(t+1) = a·y(t) + b·u(t)

**Нелинейная модель:**  
y(t+1) = a·y(t) − b·y(t−1)² + c·u(t) + d·sin(u(t−1))

Параметры модели: a = 0.9, b = 0.1, c = 0.5, d = 0.2.  
Начальная температура: y(0) = 20°C.  
Входное воздействие: u(t) = 1 (постоянный нагрев).  
Количество шагов моделирования: n = 50.

---

## Результаты

При линейной модели температура плавно снижается от 20 до установившегося значения, система устойчива и предсказуема. При нелинейной модели квадратичный член с отрицательным знаком вызывает неограниченный рост температуры по модулю — система становится неустойчивой, что требует синтеза регулятора.
