## Постановка задачи

Разработать программу на C++, моделирующую изменение температуры объекта, описываемого дифференциальным уравнением:

$$\Large\frac{dy(\tau)}{d\tau}=\frac{u(\tau)}{C}+\frac{Y_0-y(\tau)}{RC}$$

На основе этой модели получены две дискретные модели:

**Линейная:**

$$\Large y_{\tau+1}=a \cdot y_\tau + b \cdot u_\tau$$

**Нелинейная:**

$$\Large y_{\tau+1}=a \cdot y_\tau - b \cdot y_{\tau-1}^2 + c \cdot u_\tau + d \cdot \sin(u_{\tau-1})$$

---

## Параметры модели

- $a = 0.8$
- $b = 0.3$
- $c = 0.4$
- $d = 0.1$
- Начальная температура $y_0 = 20^\circ C$
- Начальный нагрев $u_0 = 5$

**Управляющий сигнал:**
- $u = 5$ для $\tau < 10$
- $u = 10$ для $\tau \ge 10$
